# Product Context: md2img

## 为什么存在
开发者和技术写作者经常需要将 Markdown 内容（如文档片段、代码示例、README 预览）转换为高质量图片，用于社交媒体分享、博客配图、演示文稿等场景。现有工具要么需要手动截图，要么样式不一致。md2img 提供了一个「所见即所得」的解决方案。

## 解决的问题
1. **样式一致性**：使用 Playwright 统一渲染，确保输出图片与 GitHub 风格一致
2. **效率**：实时预览 + 一键导出，无需截图工具
3. **可编程**：REST API 支持批量处理和集成到其他工作流
4. **灵活性**：多格式（PNG/JPEG/WebP）、可调宽度和质量、亮/暗主题
5. **AI 集成**：MCP Server 让 AI Agent 可以直接调用 md2img 能力

## Markdown 扩展支持
md2img 不仅支持标准 Markdown，还内置了丰富的扩展语法：
- **LaTeX 数学公式**：`$E=mc^2$` 和 `$$\int_0^1 f(x)dx$$`
- **Emoji 短代码**：`:rocket:` → 🚀，`:smile:` → 😄（125+ 映射）
- **脚注**：`[^1]` 引用 + `[^1]:` 定义
- **上标/下标**：`X^2^` → X²，`H~2~O` → H₂O
- **高亮**：`==text==` → <mark>text</mark>

## 用户体验目标
- 打开网页即可使用，无需注册
- 编辑器内容变动后预览即时更新（200ms 防抖）
- 导出图片清晰（2x 分辨率）
- 暗色模式完整支持
- API 文档内嵌在应用中，方便开发者查阅
- 丰富的 Markdown 扩展语法，满足技术写作需求

## 目标用户
- 开发者（生成文档截图、代码片段图片）
- 技术写作者（博客配图）
- 需要程序化生成 Markdown 图片的团队
- AI Agent（通过 MCP 协议调用）
