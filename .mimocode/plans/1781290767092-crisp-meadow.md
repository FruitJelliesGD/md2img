# UI Optimization Plan

## Goal
优化网页UI布局，解决工具栏布局混乱、模板选择器样式不统一、视觉层次不清晰的问题。

## Current Issues

### 1. Toolbar Layout
- 三栏布局 (`justify-between`) 无溢出处理
- 中间区域（TemplateSelector + ExportSettings）可能与左右区域冲突
- 无视觉分隔符区分不同功能组

### 2. Template Selector
- 当前样式像表单控件（有label、全宽select、mb-2边距）
- 不符合工具栏的紧凑内联风格
- 显示描述文字占用过多空间

### 3. Visual Hierarchy
- 深色模式背景色不统一（gray-900、#0d1117、gray-800）
- 定义了 `surface-*` 颜色令牌但未使用
- 字体不一致（3种不同的sans-serif字体栈）
- StatusBar有硬编码英文字符串

---

## Implementation Plan

### Phase 1: TemplateSelector Redesign
**File**: `frontend/src/components/TemplateSelector.vue`

**Changes**:
- 移除 `<label>` 元素和描述文字
- 添加模板图标 + 下拉箭头
- 匹配 ExportSettings 的紧凑样式
- 使用 `min-w-[120px]` 固定最小宽度

```vue
<template>
  <div class="flex items-center gap-2">
    <svg class="w-4 h-4 text-gray-500 dark:text-gray-400" ...><!-- template icon --></svg>
    <select
      v-model="selectedTemplate"
      class="min-w-[120px] px-2.5 py-1.5 text-sm border border-gray-300 dark:border-gray-600 rounded-lg bg-white dark:bg-gray-800 focus:outline-none focus:ring-2 focus:ring-primary-500"
    >
      <option value="">Default</option>
      <option v-for="t in templates" :key="t.id" :value="t.id">{{ t.name }}</option>
    </select>
  </div>
</template>
```

### Phase 2: Toolbar Layout Overhaul
**File**: `frontend/src/components/Toolbar.vue`

**Changes**:
1. **图标按钮**: Theme/Locale 切换改为纯图标按钮（移除文字）
2. **视觉分隔符**: 在功能组之间添加 `border-l` 分隔线
3. **溢出处理**: 添加 `overflow-x-auto` 和隐藏滚动条
4. **分组结构**:
   - Left: Logo + Title
   - Center-left: Theme + Locale (icons only)
   - Center: TemplateSelector + ExportSettings
   - Right: Download + Copy + API Docs

```vue
<template>
  <div class="flex items-center justify-between px-4 py-2.5 bg-white dark:bg-surface-900 border-b border-gray-200 dark:border-gray-700 shadow-toolbar">
    <!-- Left: Logo -->
    <div class="flex items-center gap-2">
      <div class="w-7 h-7 rounded-lg bg-primary-600 flex items-center justify-center">
        <svg .../>
      </div>
      <h1 class="text-lg font-bold text-gray-900 dark:text-white">md2img</h1>
    </div>

    <!-- Center: Controls -->
    <div class="flex items-center gap-3">
      <!-- Theme & Locale (icon-only) -->
      <div class="flex items-center gap-1 border-r border-gray-200 dark:border-gray-700 pr-3">
        <BaseButton size="sm" variant="ghost" aria-label="Toggle theme">
          <svg .../> <!-- sun/moon icon -->
        </BaseButton>
        <BaseButton size="sm" variant="ghost" aria-label="Toggle language">
          <svg .../> <!-- globe icon -->
        </BaseButton>
      </div>

      <!-- Template & Export -->
      <div class="flex items-center gap-3">
        <TemplateSelector />
        <ExportSettings ... />
      </div>
    </div>

    <!-- Right: Actions -->
    <div class="flex items-center gap-2">
      <BaseButton variant="primary" ...>Download</BaseButton>
      <BaseButton variant="secondary" ...>Copy</BaseButton>
      <BaseButton variant="ghost" size="sm" ...>API</BaseButton>
    </div>
  </div>
</template>
```
